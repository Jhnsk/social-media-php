# Social Media PHP

Projeto de uma rede social desenvolvido com PHP Orientado a Objetos, seguindo a arquitetura MVC e utilizando PDO para acesso ao banco de dados.

O objetivo deste projeto foi praticar conceitos importantes do desenvolvimento web backend, como autenticação de usuários, manipulação de banco de dados, relacionamentos entre tabelas, upload de arquivos, AJAX e organização de código.

## Funcionalidades

* Cadastro de usuários
* Login e logout
* Sistema de sessões
* Criação de posts
* Upload de imagens
* Feed de publicações
* Curtidas em posts
* Comentários
* Sistema de seguidores
* Perfil de usuário
* Messenger privado entre usuários
* Atualização de mensagens utilizando AJAX
* Exibição da última mensagem nas conversas

## Tecnologias Utilizadas

* PHP
* MySQL / MariaDB
* PDO
* JavaScript
* AJAX
* HTML5
* CSS3

## Estrutura do Projeto

O projeto segue o padrão MVC (Model-View-Controller), separando responsabilidades entre:

* Models: acesso e manipulação de dados
* Views: interface do usuário
* Controllers: regras de negócio e fluxo da aplicação

## Instalação

1. Clone o repositório.
2. Crie um banco de dados MySQL/MariaDB.
3. Copie o arquivo `Database.example.php` para `Database.php`.
4. Configure as credenciais do banco de dados.
5. Importe a estrutura das tabelas.
6. Execute o projeto utilizando XAMPP ou outro servidor PHP.

## Objetivo do Projeto

Este projeto foi desenvolvido com fins de estudo e prática de desenvolvimento web utilizando PHP puro, com foco em compreender os conceitos que frameworks modernos, como Laravel, abstraem no dia a dia.
